<div align="center">
# ComfyUI-Z-Image-Turbo  
  
Comfy UI nodes for Z-Image-Turbo to run native weights

</div>

✅ Install on Comfy:
```
cd custom_nodes
git clone https://github.com/tpc2233/ComfyUI-Z-Image-Turbo.git

cd ComfyUI-Z-Image-Turbo
pip install modelscope
pip install git+https://github.com/huggingface/diffusers
pip install -r requirements.txt
```


✅ Comfy Workflow:
```
https://github.com/tpc2233/ComfyUI-Z-Image-Turbo/blob/main/workflow.json
```

✅ Autodownload the models:

It will download the models first time you run workflow

It will be downloaded to:
ComfyUI/models/diffusers/Z-Image-Turbo



✅ Run:

24Gb VRAM usage on 2048x2048
